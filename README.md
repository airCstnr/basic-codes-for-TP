
# basic-codes-for-TP

Blocs unitaires de codes pour les TP de 3App à l'ESISAR.

## Contenu

Morceaux de code pour les TP, pour ne pas perdre du temps à réécrire les fonctions
standard à chaque TP.

## Cours concernés

+ [CS357](/C) - C
+ [CE311](/VHDL) - VHDL
+ [CE317](/Archi) - Architecture (Nucléo)
